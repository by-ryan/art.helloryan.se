Code is Art!

art is a collection of open-source projects, mainly C++ libraries useful for building applications. E.g. libart-json for JSON manipulation and the libart-seafire collection implementing an HTTP/1.1 server based on ASIO.

libart-seafire is named in honor of my beloved dog, Sefyr.
